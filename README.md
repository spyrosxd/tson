# TSON

**Convert JSON to TypeScript interfaces instantly.**

---

## 🚀 About

TSON is a free, fast, and simple tool for developers to quickly generate clean TypeScript interfaces from any JSON data.  
Just paste your JSON and get ready-to-use TypeScript typings in seconds — no manual typing required!

---

## 💡 Features

- Paste raw JSON and generate accurate TypeScript interfaces  
- Support for nested objects and arrays  
- Optional properties detection for nullable fields  
- Clean, readable, and prettified output  
- Copy to clipboard with one click  
- Fully client-side — no backend or signup needed

---

## 🎯 Why use TSON?

- Save time hand-writing interfaces from API responses  
- Avoid errors in manual typings  
- Easily integrate generated interfaces into your projects  
- Perfect for frontend and full-stack devs working with TypeScript

---

## 🛠️ Usage

1. Open the app: [https://tson.vercel.app](https://tson.vercel.app)
2. Paste your JSON into the input editor  
3. Instantly see generated TypeScript interfaces  
4. Copy the output and paste it into your codebase
